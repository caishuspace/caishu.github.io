逻辑回归部分

L(y-hat,y) 损失函数

截图里面，可以使用平方差模型作为损失函数，但是会出现两个问题，就是if y=1 和y=0 部分 【会产生非凸优化函数的问题】，故选择设定函数（方框标注的L函数）作为损失函数

J（w,b）代价函数

在优化逻辑回归函数时，取w,b最优，使得代价函数最小。
